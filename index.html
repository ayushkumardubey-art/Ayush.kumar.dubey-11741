About Me

Hello! I am a student who loves building things for the web. I started my journey exploring HTML & CSS, and now I am diving deeper into JavaScript.

This portfolio is a collection of my Academic Lab Records, Practical Assignments, and my Mini Projects. I built this site myself to practice what I learn in class.

How I Built This

This portfolio was constructed using the following core HTML elements:

<div>

Used as the main building block to group content and create the layout cards.

<span>

Used for inline styling, specifically for the "Typewriter" text effect and highlighting colors.

<nav>

The semantic tag used for the top navigation bar containing the links.

<section>

Used to divide the page into distinct areas: About, Work, and Contact.

<a>

Anchor tags used for navigation links and the buttons to download PDFs.

<button>

Used specifically for the Dark/Light mode toggle switch in the top right.
